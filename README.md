# c64-font-printer
print C64 fonts as ASCII pixels.

For questions and suggestion write an email to
Dirk Jagdmann <doj@cubic.org>
visit also http://d0j.de and http://llg.cubic.org

## example

```
./c64-font-printer 'POKE53280,0'
```
